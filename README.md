# Ruby TF Outputs

This gem takes your terraform outputs and imports them into ruby. 

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'tfoutput'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tfoutput
